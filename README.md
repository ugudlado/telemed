Video chat poc with 
> React
> Twilio apis

Running the code in local machine

> Only front end
cd client
npm install 
npm start

> Only Api
cd server
npm install
npm start

> both Api and frontend together
cd client
npm install
npm run postinstall

cd server
npm install
npm run postinstall
npm start
